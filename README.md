# 基础算法的基本训练
对于不同开发领域的程序员来说，编程语言可能不同，技术领域、业务领域也可能不同，但是，
扎实的计算机科学基础应该是每个程序员都应该掌握的。包括但不限于操作系统，编译原理，算法与数据结构等。

我也是在两年左右的工作中，面试中总结到的: 扎实的基础是每一个优秀程序员的硬实力，也是进入优秀企业面试成功的基本保障。

下面是我在工作学习过程中，积累下来的基础算法训练。
